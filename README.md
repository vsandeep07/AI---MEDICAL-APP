# AI Medical Diagnosis System

## Live Demo
Try the application here:
https://ai---medical-app-pqxzg7zbbejxdnst8vtkez.streamlit.app/

## Project Description
This is an AI-powered medical diagnosis system that predicts possible diseases based on symptoms provided by the user.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Streamlit(using chatgpt)

## Features
- Disease prediction
- Diet recommendations
- Medication suggestions
- Workout recommendations(in progress)

## platform used 
- jupyter notebook
- vs code
